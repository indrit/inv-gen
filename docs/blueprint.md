# **App Name**: InvoiceFlow

## Core Features:

- Invoice & Estimate Creation: Intuitive forms for generating customizable invoices and estimates with all fields, including tax, being fully editable by the user.
- PDF Generation & Download: Secure generation and download of professional invoice and estimate documents in PDF format. Two downloads are free without login, followed by three additional downloads after Google or Apple login.
- User Authentication: Seamless user login functionality via Google and Apple accounts to unlock extended usage and personalized features.
- Monetization - Donation Prompt: A friendly 'Buy Me a Coffee' donation prompt presented to users after specific usage tiers, with intelligent tracking for declining the offer up to two times.
- Monetization - Subscription Management: Subscription handling for a premium plan ($6.99/month) which removes ads and provides automatic invoice numbering and tracking capabilities.
- Progressive Web App (PWA) Installability: The application will be installable as a Progressive Web App (PWA) icon on mobile devices, offering a native-like user experience.
- Integrated Blog with AI Content Tools & CMS: A dedicated, SEO-friendly blog section for publishing articles and content, powered by a user-friendly CMS backend. Includes an AI tool to assist in generating SEO-optimized blog post outlines, keywords, and initial draft content.

## Style Guidelines:

- Primary color: A sophisticated, deep indigo (#262E7A) to convey professionalism and reliability in a light scheme.
- Background color: A very subtle, cool light gray (#F2F3F7) derived from the primary hue, ensuring maximum readability and clarity for forms and text.
- Accent color: A bright, vibrant cyan (#4DD1FF) used for calls to action, highlights, and to provide a refreshing contrast to the primary color.
- Headlines will use the 'Space Grotesk' sans-serif font for a modern, precise, and tech-forward feel. Body text, form labels, and other content will use the 'Inter' sans-serif font, chosen for its excellent legibility and neutral professionalism, ideal for data-heavy applications.
- Use clear, concise, and professional line-art icons that complement the clean interface, focusing on functionality and ease of recognition.
- Implement a clean, spacious, and highly responsive layout, prioritizing mobile-first design with a strong emphasis on logical content flow and ease of form completion across all device sizes.
- Subtle, smooth, and purposeful animations for user feedback (e.g., button presses, form submission, PWA install prompts), ensuring a fluid and reassuring user experience without distractions.
